# 操作方法

scaffolding 段階のため、現状は移動と scene 進行だけが実装されている。

## Title scene

| キー | アクション |
|---|---|
| `Enter` / `Space` | Battle scene へ進む |

## Battle scene (Player 操作)

矢印キーと WASD は同義 (4 方向同時押し可)。

| キー | アクション |
|---|---|
| `→` / `D` | 右へ移動 |
| `←` / `A` | 左へ移動 |
| `↑` / `W` | 奥へ移動 (Z 方向) |
| `↓` / `S` | 手前へ移動 (Z 方向) |

ジャンプ / 攻撃 / ガード / カメラ操作などは未実装。
